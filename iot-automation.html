The Smart Home Brain

A privacy-first, fully local home automation ecosystem. Centered around Home Assistant and dedicated hardware control layers.

System Architecture

Hardware Controller

The physical foundation ensuring reliable, low-latency execution of automations.

Compute: Intel NUC

Dedicated small-form-factor PC providing ample processing power for real-time logic without the overhead of heavy servers.

Radio: Nabu Casa Connectivity

Dual USB radio stick integration providing native communication with separate Zigbee and Z-Wave mesh networks.

The Brain (Software)

Run by Home Assistant. The open-source platform that unifies all devices into a single control plane.

  • Local Control: All logic is processed on-device. No cloud dependencies means automations work even when the internet is down.
  • Complex Logic: Utilizing Node-RED alongside native automations to handle multi-conditional scenarios.
  • Unified Dashboards: Custom UI for tablet and mobile control of lights, climate, and media.

Zigbee

Used for battery-powered sensors (Temp, Motion) and lights. Fast mesh networking.

Z-Wave

Used for critical infrastructure like door locks, thermostats, and in-wall switches. Better range and reliability.

WiFi (Local)

Used for high-bandwidth devices like cameras and media players, confined to a separate IoT VLAN.
