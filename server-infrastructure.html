Server Infrastructure

The logical layer powering the hardware. A blend of virtualization, containerization, and high-performance computing.

The Software Stack

Media Pipeline

Hosted primarily on the Dell R730 (Unraid). This node manages the entire lifecycle of media acquisition and delivery.

  • Plex Media Server: The core application for streaming content locally and remotely.
  • Automation: Full *Arr stack integration for seamless content management.
  • Transcoding: Hardware accelerated by Nvidia GPUs for buffering-free 4K streams.

Storage & Security

Powered by the Dell R730XD (Unraid). This High-Density storage server acts as the secure vault for data and physical security.

  • Unraid Array: 12-bay configuration providing massive redundant storage capacity.
  • Network Video Recorder: Dedicated 24/7 recording endpoint for security cameras.
  • Docker Laboratory: A sandbox environment for spinning up experimental services and tools.

Artificial Intelligence & Compute

Workstation Node (Ubuntu)
Equipped with an Nvidia Quadro RTX 5000. This system handles the heavy lifting for AI model training and complex data processing tasks.

Inference Node (R730)
Leveraging dual Nvidia Tesla P40s (48GB Total VRAM) to host and query Large Language Models (LLMs) locally, ensuring data privacy and low latency.
